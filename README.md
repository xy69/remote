Tested on Arm Debian 12.

## How to use

```sh
sudo apt install -y curl && sudo sh -c "$(curl -fsSL https://raw.githubusercontent.com/xy69/remote/main/install.sh)"
```

### Recommendation

```sh
cd /opt/remote && ./start.sh
```
